# Rizka Audio Padang

Website katalog produk toko audio mobil yang berlokasi di **Padang Pasir IV No.28A, Padang Barat, Kota Padang**.

Menyediakan berbagai perlengkapan audio mobil: speaker, subwoofer, power amplifier, tweeter, head unit, hingga box subwoofer. Tersedia merek-merek ternama seperti Pioneer, Nakamichi, Kicx, Blaupunkt, dan lainnya.

Dibangun dengan **Vue 3 + Vite + Tailwind CSS v4**. Data produk statis, tidak perlu backend.

## Cara menjalankan

```bash
npm install
npm run dev
```

Buka http://localhost:5173 di browser.
